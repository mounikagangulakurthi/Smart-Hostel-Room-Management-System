**Smart Hostel Room Management & Security System
Overview**

This project is an IoT-based hostel room management and security system developed using ESP32, RFID, PIR sensor, DHT sensor, and OLED display.

**Features**
RFID-based secure access
Intruder detection using PIR sensor
Temperature and humidity monitoring
OLED display status updates
LED indicators and alarm system
**Components Used**
ESP32
MFRC522 RFID Module
PIR Sensor
DHT Sensor
OLED Display
LEDs
Buzzer
**Working**

The system authenticates users through RFID cards. Unauthorized access triggers an alarm. Environmental conditions are monitored and displayed on the OLED screen.
